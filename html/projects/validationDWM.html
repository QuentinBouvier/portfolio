Projet de validation de la formation DWM8

Consulter Git repo

Projet de validation de la formation Développeur Web de It-akademy, réalisé en 15h.

Thème proposé: un magasin de musique. Thème choisi: Ma collection de guitares

La demande:

  • Un CRUD (create, read, update, delete) codé grâce à Laravel 5, utilisant une DB MySql
  • La base de donnée doit contenir au moins une relation simple <1:N>
  • La base de donnée doit contenir au moins une table intermédiaire pour relations <N:N>
  • Le résultat doit être responsive jusqu'à 800px de largeur
  • Le projet doit être rendu sur gitHub dans la branche "production" d'un repository tout neuf

Features:

  • Max responsiveness: Ok jusqu'à ~750px
  • Ajout, édition suppression de guitares
  • Ajout de propriétés simples et complexes (Marques, Micros pour guitares électriques, Couleurs, Bois, types de guitares).
  • Affichage des guitares sur 2 lignes de tableau avec déploiement de la seconde ligne.
  • Déploiement des panneaux d'ajout de propriété.
  • Gestion des propriétés via des requètes ajax.

Technos du rendu:

  • HTML/css flexbox
  • jQuery dom manipulation
  • jQuery.ajax()
  • PHP objet
  • Laravel 5.5:
    • Routes post/get, web et api
    • Eloquent: all, find, where, save, attach, detach, delete, belongsTo, belongsToMany
    • Model, controller
    • php artisan (create controllers, models, migrations, generate app key)
    • Blade templating
  • MySql
  • Vagrant, ubuntu xenial, php7.0, apache2, mysql, composer
