StoryStation, le site dont vous être le héros

Sources server Sources Webapp Sources systèmes embarqués Intitulé et rendu

Développement en équipe de 6 d'un projet à sujet libre en 3 semaines, avec l'utilisation d'un raspberry 3+ et d'un kit de conception électronique pour débutant.

Prototype d'un site web proposant des histoires dont vous êtes le héros, sortes de petites aventures textuelles à embrachements. Les étapes de l'histoire peuvent parfois demander l'intervention du ou des joueurs sur des modules électroniques interactifs.

Techno:

  • Python
    • flask
    • gpio
    • gevent
  • mongo
  • angular
