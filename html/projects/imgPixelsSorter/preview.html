Image pixels Sorter

Télécharger Processing 1.5.1

Un programme écrit avec Processing permettant de reclasser les pixels d'une image sur les axes du plan en fonction de leur teinte et luminosité

J'ai utlisé ce logiciel lors de mes études aux beaux arts pour produire des images de grande tailles d'après des photos, en vue d'un accrochage en espaces d'exposition.

Technologies:

  • Processing 1.5.1
  • Poo, classes
  • Typage
  • App graphique 2D
  • Algorithme QuickSort

Fractales

Un programme écrit avec processing qui permet de dessiner les étapes d'une structure fractale.

Attention: Programme pouvant utiliser un max de ram. Lent à partir de quelques étapes.

Utiliser les flèches gauche et droite pour avancer ou reculer d'une étape, haut et bas pour zoomer.
