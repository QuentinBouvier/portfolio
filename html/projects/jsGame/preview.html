I WANNA BE THE BORING GRID GAME JS version

Jouer

IWBTBGG est un simple jeu de plateau qui implique The Guy, des cerises (inoffensives cette fois) et des zombies.

Le personnage doit attraper 15 cerises pour gagner. S'il est touché par un zombie, la partie est perdue

Le personnage est dirrigé grâce aux touches du clavier sur une grille de x*x cases (max 34*15), taille définie par le joueur.

Saurez vous survivre aux déplacements chaotiques de vos énemmis, et aux érrances aléatoire de votre objectif ?

Bonne chance !

Technologies :

  • Local storage
  • jQuery
  • Inputs
  • Gestion des affichages
  • Formulaires JS
  • Css flexbox

Spoilers :

  • Le nombre original de zombies est défini par le nombre total de cases. 1 pour 75 cases
  • Les zombies ont 75% de chances de se déplacer de 1 cases dans une direction aléatoire par tour.
    Ils ont une chance supplémentaire de 20% de se déplacer dans une autre direction, de 1 à 3 cases
  • La cerise a 10% de chances de se déplacer de 1 à 6 cases dans une direction aléatoire
  • Après un nombre de pas prédéfini, un zombie supplémentaire apparait:
    • Facile: tous les 70 pas
    • Moyen: tous les 50 pas
    • Difficile: tous les 25 pas
