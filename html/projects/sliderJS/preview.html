Slider jQuery

Aperçu

Un slider JS interactif, proposant un défilement horizontal dans les deux sens.

Le slider est multi-instances, et peut gérer un nombre illimité d'images

Construit comme un plugin js autonome, le slider s'intègre dans une page contenant un containeur particulier.

Technologies :

  • jQuery
  • jQuery animate()
  • Inputs
  • Manipulation Dom
  • Css flexbox
